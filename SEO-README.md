# Joad Yimel González Tovar - CV/Portfolio

Award-winning dark brutalist CV/portfolio landing page with bilingual support and advanced SEO optimization.

## 🌐 Bilingual Support (EN/ES)

### Features
- **Automatic language detection** based on browser preferences
- **Manual language toggle** in the navigation bar
- **Persistent language selection** stored in localStorage
- **Dynamic content translation** for all sections
- **SEO optimization** for both languages

### Language Toggle
Click the language button (EN/ES) in the top navigation to switch between English and Spanish. Your preference will be remembered across sessions.
